# Deep Learning-Based microRNA Target Prediction Using Experimental Negative Data

### :wrench: To install python dependencies:
```bash
$ sh setup.sh
```

### :file_folder: To download training and test datasets:
```bash
$ sh download.sh
```

### :mag: To see usage:
```bash
$ python main.py --help
```
